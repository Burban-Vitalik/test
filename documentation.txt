git status - перевірити, що зараз відбувається
git rm --cached file.html - перестаємо слідкувати за файлом
У файл .gitignore ми прописуємо файли, папки які не потраплять в git
git branch -D test - ми видаляємо гілку
git checkout test - переключаємось на гілку
git checkout -b new - створюю і одразу переключаюсь на нову гілку
git merge test - ми об'єднуємо 2 гілки
